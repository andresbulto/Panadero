# 🥖 El Diario del Panadero

Aplicación personal para registrar y gestionar recetas de pan y pizza artesanal.

## Recetas incluidas
- 🥖 **Baguette Tradition** — Tradición francesa, ciclo de 60h con masa vieja
- 🍕 **Pizza Napolitana** — Poolish + Ooni, hidratación 72%
- 🫓 **Pa de Coca** — Focaccia catalana con bassinage

## Funcionalidades
- ⚖️ Escalado automático de ingredientes por cantidad
- ⏱ Temporizadores integrados en cada paso
- 🌡 Ajuste de fermentación por temperatura de cocina
- ✅ Checklist de pasos con progreso
- ✍️ Diario de intentos con fotos y valoraciones (1–5 ⭐)
- 📅 Vista calendario semanal con todos los pasos programados
- 🏆 Historial de hornadas con gráfico de evolución
- 🛒 Lista de compra generada automáticamente
- 💾 Todo guardado localmente en el navegador (localStorage)

## Despliegue

### Vercel (recomendado)
1. Sube esta carpeta a GitHub
2. Importa el repositorio en [vercel.com](https://vercel.com)
3. Deploy automático — sin configuración adicional

### Local
Abre `index.html` directamente en el navegador. No necesita servidor.

## Tecnología
HTML + CSS + JavaScript puro. Sin dependencias externas salvo Google Fonts.
Todo el estado se guarda en `localStorage` del navegador.
